# Pixiv 图片下载器

一个基于 CustomTkinter 的华丽界面 Pixiv 图片下载工具，支持批量下载作者作品。

## ✨ 特性

- 🎨 **华丽界面**: 基于 CustomTkinter 的现代化深色主题界面
- 🚀 **多线程下载**: 支持自定义线程数，提高下载效率
- 📊 **实时进度**: 显示下载进度条和详细统计信息
- 📝 **下载日志**: 实时显示下载状态和错误信息
- ⚙️ **灵活配置**: 可调整线程数、下载间隔等参数
- 🎯 **批量下载**: 一键下载作者所有作品
- 🛑 **可控停止**: 支持随时停止下载任务

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

```bash
python main.py
```

## 📖 使用说明

1. **输入作者ID**: 在"作者ID"输入框中输入要下载的Pixiv作者ID
2. **设置Cookie**（可选）: 如果需要，可以输入您的Pixiv Cookie
3. **代理设置**（可选）:
   - 填写手动代理列表；
   - 或开启 Quake 自动爬取，输入 `API Key` / `Cookie`；
   - 系统会先校验代理可用性，再加入代理池。
4. **调整参数**: 根据需要调整线程数和下载间隔（代理模式线程自动限制在 3~6）
5. **开始下载**: 点击"开始下载"按钮开始下载
6. **查看进度**: 实时查看下载进度和日志信息
7. **停止下载**: 如需停止，点击"停止下载"按钮

## 📁 下载位置

下载的图片默认保存在 `Sakura_Downloads/{作者ID}/` 目录下。

## 🔧 技术栈

- **GUI框架**: CustomTkinter 5.2+
- **HTTP请求**: Requests 2.31+
- **图像处理**: Pillow 10.0+

## ⚠️ 注意事项

1. 请遵守Pixiv的使用条款，仅下载您有权限访问的内容
2. 建议合理设置下载间隔，避免对服务器造成过大压力
3. Cookie可能会过期，如遇到下载失败，请更新Cookie
4. 请勿用于商业用途
